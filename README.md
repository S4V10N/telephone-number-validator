# US Telephone Number Validator

This app quickly checks whether a phone number matches valid US formats. Users enter a number, click the check button, and instantly see whether the number is valid or invalid based on strict formatting rules. It provides clear, accurate feedback while keeping the interface simple and easy to use.

## Overview

### The challenge

-   Users are able to:
-   Enter a US phone number in various common formats
-   Validate the number with a single click
-   Instantly see whether the number is valid or invalid
-   Clear the results to check another number
